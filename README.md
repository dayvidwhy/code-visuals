Application makes use of:
* Svelte
* SvelteKit
* TailwindCSS

## Install
Clone the repository and start development mode.
```bash
git clone git@github.com:dayvidwhy/code-visuals.git
cd code-visuals
npm i
npm run dev -- -open

```

Project uses NVM for [node](https://github.com/nvm-sh/nvm) version. 
```bash
# Set node to 21
nvm use
```
